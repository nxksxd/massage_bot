## 🚀 Как использовать

### Первый запуск
```bash
# 1. Установка зависимостей
pip install -r requirements.txt

# 2. Конфигурация
cp .env.example .env
# Отредактируем .env - добавим токен и ID таблицы

# 3. Добавим credentials.json
# (скачиваем с Google Cloud Console)

# 4. Запуск
python bot.py
```

### Все параметры оптимизации в одном месте
Файл: `config.py`
```python
GOOGLE_SHEET_CACHE_TTL = 300           # Кеш на 5 минут
GOOGLE_SHEETS_MAX_RETRIES = 3          # 3 попытки
GOOGLE_SHEETS_TIMEOUT = 30             # 30 сек таймаут
GOOGLE_SHEETS_RETRY_DELAY = 1          # Базовая задержка 1 сек
LOG_LEVEL = "INFO"                     # Уровень логирования
```
